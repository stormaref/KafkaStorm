# Security Policy

## Supported versions

| Version | Supported          |
| ------- | ------------------ |
| 10.x    | :white_check_mark: |
| < 10    | :x:                |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, use [GitHub Security Advisories](https://github.com/stormaref/KafkaStorm/security/advisories/new) to report the issue privately. You will receive a response as soon as possible.

Include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)
